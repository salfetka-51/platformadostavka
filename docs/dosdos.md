### Введение

***Платформа для доставки*** - это программное решение предназначенное для оформление, обработки и отслеживания заказов.

Платформа позволяет пользователям:
- выбрать адрес доставки
- отслеживать заказ
- оформлять заказы онлайн
- получать уведобления о статусе заказа

**Основная цель**: упростить путь между продавцом и клиентом

### Примеры использования 
1. При нажатии на кнопку **"Регистрация"** пользователь создает новый аккаунт
2. При вводе логина и пароля пользователь заходит в свой аккаунт
3. Прии выборе определенного товара он автоматически добавляется в корзину
4. При нажатии на кнопку **"Подтвертить заказ"** заказ автоматически переходит в обработку
5. При нажатии на кнопку **"Категории"** пользователь может выбрать определенную категорию товаров
