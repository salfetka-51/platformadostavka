### Введение

***Платформа для доставки*** - это программное решение предназначенное для оформление, обработки и отслеживания заказов.

Платформа позволяет пользователям:
- выбрать адрес доставки
- отслеживать заказ
- оформлять заказы онлайн
- получать уведобления о статусе заказа

**Основная цель**: упростить путь между продавцом и клиентом

### Примеры использования 
1. При нажатии на кнопку **"Регистрация"** пользователь создает новый аккаунт
2. При вводе логина и пароля пользователь заходит в свой аккаунт
3. Прии выборе определенного товара он автоматически добавляется в корзину
4. При нажатии на кнопку **"Подтвертить заказ"** заказ автоматически переходит в обработку
5. При нажатии на кнопку **"Категории"** пользователь может выбрать определенную категорию товаров 
6. При нажатии на кнопку **"Отследить заказ"** открывается карта с текущим местоположением курьера
7. При выборе способа оплаты **"Банковской картой"** появляются поля для ввода платежных данных
8. При нажатии на кнопку **"Повторить заказ"** в истории все товары из прошлого заказа добавляются в корзину
9. При изменении количества товаров в корзине итоговая сумма заказа пересчитывается автоматически
10. При нажатии на кнопку **"Избранное"** пользователь может сохранять понравившиеся товары

### Технические требования

Для корректной работы платформы для доставки необходимо обеспечить следующие технические условия:

| Компонент | Требование |
| :--- | :--- |
| **Операционная система** | Windows 10/11, macOS 11+, Linux (Ubuntu 20.04+) |
| **Браузер** | Google Chrome 90+, Mozilla Firefox 88+, Safari 14+, Microsoft Edge 90+ |
| **Интернет-соединение** | Скорость не менее 10 Мбит/с |
| **Разрешение экрана** | Минимум 1280x720 пикселей |
| **Дополнительно** | Включенный JavaScript, поддержка cookie, геолокация (для отслеживания) |

**Поддерживаемые мобильные устройства:**
- iOS 14.0 и выше
- Android 10.0 и выше
